Speaktionary

A dictionary you can speak to! I used Apple's AVFoundation framework for recording and Speech framework to process the audio. Additionally I used CoreData to store the results in a SQL database and SwiftyJSON to consume the Oxford Dictionaries API.

This project is open-source so you can view the source code over at GitHub.

The Product

Speaktionary is, as can be deduced from the name, a dictionary app that has to do with speech. It's super easy to use. In case you forget a word you looked up, you can look it up in the saved searches where every word is automatically saved. You can as well, of course, delete saved words from the list.

The Idea

I came up with the idea when I was doing English homework assignments. I though it would be handy to have the complete Oxford dictionary accessible in an instant.

The Process

It was quite a challenge to figure out how to live record and process audio. I ended up using a memory buffer to store the audio temporarily. After the user had finished recording, the audio is transformed into text. Finally the defintion is fetched from the Oxford API. After digging trough much JSON (thanks, SwiftyJSON!) the definition is presented to the user.
